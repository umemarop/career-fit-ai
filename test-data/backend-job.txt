Backend Developer

Responsibilities:
- Design and develop RESTful APIs using Node.js and Express
- Build database models and queries using PostgreSQL
- Implement authentication and authorization
- Write clean and testable backend code
- Handle validation, error handling, and logging
- Work with Docker-based local development environments
- Collaborate with frontend developers to support product features

Required skills:
- Node.js
- Express
- TypeScript
- PostgreSQL
- REST APIs
- Authentication
- Git

Preferred skills:
- Prisma
- Docker
- Redis
- AWS
- Unit testing
- CI/CD pipelines

This role is ideal for a developer who understands backend fundamentals and wants to work on production-grade API systems.
